# SecureFlow AI

Plateforme multi-agents pour **auditer**, **concevoir** et **analyser** des projets logiciels — hackathon Band of Agents 2026.

## Modes

| Mode | Agents | Usage |
|------|--------|-------|
| A — Security Audit | 5 | Audit sécurité d'un projet existant |
| B — Full Dev Pipeline | 6 | Conception + dev d'un nouveau projet |
| C — Rapport PDF | 5 | Analyse + livrable PDF professionnel |
